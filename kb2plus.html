Liste Menu / Navigation

Une barre de navigation est une section essentielle d'un site web qui permet aux utilisateurs de naviguer entre différentes pages ou sections d'une même page. Généralement, elle est placée en haut de la page, mais elle peut également se trouver sur le côté ou ailleurs, selon la conception du site.

Dropdown Menu

Un menu dropdown (menu déroulant) dans une barre de navigation est un élément interactif qui affiche une liste d'options supplémentaires lorsqu'on passe la souris dessus ou qu'on clique dessus.

Box-Shadow

Tooltips

Un tooltip (ou info-bulle) est un petit texte d’information qui apparaît lorsqu’un utilisateur passe la souris sur un élément ou clique dessus. Il fournit des informations supplémentaires sans encombrer l'interface principale.

Logo Web

Une favicon est une petite icône affichée à côté du titre d'une page web dans l'onglet d'un navigateur. Elle aide les utilisateurs à identifier visuellement un site parmi plusieurs onglets ouverts ou dans les favoris enregistrés.
